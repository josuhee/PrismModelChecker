# Prism Model Checker
Create Prism model with corpus  
<br/>  

### ✔ Execution method
```
git clone https://github.com/josuhee/PrismModelChecker.git
cd PrismModelChecker/src
python main.py
```
**Output Path** : file/result.prism  
<br/>
